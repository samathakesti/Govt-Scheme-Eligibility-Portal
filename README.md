# 🇮🇳 Government Scheme Eligibility Finder & Application Portal

A simple web application to check eligibility for government schemes and apply for them.

---

## 🚀 Features
- Check eligibility using Age, Income, Category  
- Sections for Farmer, Student, Women, Pension  
- Apply option for schemes  

---

## 🛠️ Technologies
- HTML, CSS, JavaScript  

---

## 🌐 Live Demo
🔗 https://samathakesti.github.io/Govt-Scheme-Eligibility-Portal  

---

## ⚙️ How to Use
1. Enter your details  
2. Click "Check Eligibility"  
3. View schemes  
4. Apply through category pages

---

## 💡 Example Schemes Included
- PM Awas Yojana  
- Student Scholarships  
- Farmer Support Schemes  
- Pension Schemes
